# 真实图片放置说明

可将团队自己的调研照片、PPT 导出的地铁照片、运行时刻表截图或系统截图放到本目录，文件名保持如下即可被系统自动使用：

- `metro-station.jpg`：地铁站台或列车照片。
- `dispatch-center.jpg`：调度中心或监控大屏照片。
- `timetable-sheet.jpg`：运行时刻表或 Excel 截图。
- `team-research.jpg`：团队调研照片。
- `metro-line-map.jpg`：轨道线路图或抽象线网图。

如果图片不存在，页面会自动显示内置占位视觉，不会报错或空白。

请不要放入真实机构 Logo、敏感内部数据、个人隐私信息或未经授权的素材。
